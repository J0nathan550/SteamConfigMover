# Steam Config Mover

A small browser-based tool to copy Steam userdata (configs, settings, cloud save cache) from one local Steam account to another.

Mainly useful for games like **Dota 2**, where you want to move your configs, keybinds, or settings from an old account to a new one without redoing everything by hand.

## Browser support

Requires the File System Access API, so it only works in **Chrome**, **Edge**, or other Chromium-based browsers. Not supported in Firefox or Safari.
