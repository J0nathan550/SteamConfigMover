Steam Config Mover

Copy a Steam profile's local userdata (cloud saves cache, screenshots, config, shader/controller settings) from one account to another. With a zip backup taken first.

Heads up: your browser doesn't support the File System Access API, so this tool can't read or write local folders here. Use a recent version of Chrome or Edge, and open this page over https:// or http://localhost (not by double-clicking the HTML file, if your browser blocks the API on file://).
